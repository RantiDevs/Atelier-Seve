Build a luxury beauty and esthetics studio website called "Atelier Sève" using React + Vite + TailwindCSS with an Express.js backend. This should look like a $20,000 award-winning editorial website inspired by dirtylinestudio.com, illustra aesthetics, and high-fashion beauty brands.

TECH STACK:
- React + Vite (frontend)
- Express.js (backend — same Replit project, port 3001)
- TailwindCSS
- GSAP (ScrollTrigger, SplitText)
- Three.js
- Framer Motion
- Embla Carousel

BACKEND SETUP (server.js):
- Express server on port 3001
- POST route: /api/chat
- Accepts: { messages: [...] }
- Calls Groq API:
  URL: https://api.groq.com/openai/v1/chat/completions
  Model: llama3-70b-8192
  Auth: Bearer process.env.GROQ_API_KEY
- System prompt injected server-side:
  "Sei Chiara, una consulente di bellezza italiana raffinata e premurosa per Atelier Sève. Il tuo tono è caldo, elegante e rassicurante. Raccogli: nome del cliente, trattamento desiderato (viso, ciglia, corpo, epilazione, labbra), data e ora preferita. Parla sempre in italiano. Alla fine conferma la prenotazione con grazia."
- Returns response text
- CORS enabled
